# 🌸 HƯỚNG DẪN TẢI FILE APK - MẸ & BÉ 💕

Chào anh! Em hiểu là anh đang không nhìn thấy file **`Install_Me_Va_Be.apk`** xuất hiện ở cột danh sách thư mục (File Explorer) bên trái màn hình. 

Dưới đây là nguyên nhân và 2 cách cực kỳ đơn giản để anh tải file APK về cài đặt:

---

### 1. Nguyên nhân anh không thấy file APK ở cột bên trái:
* **Cơ chế ẩn của AI Studio:** Trình quản lý thư mục (File Explorer) của trang web Google AI Studio được lập trình để **tự động ẩn các file cài đặt dạng nhị phân cực nặng như `.apk`** để tránh làm rối mắt người dùng soạn thảo code. 
* Do đó, dù em đã build thành công và đặt file `Install_Me_Va_Be.apk` ở thư mục gốc, hệ thống cũng **không hiển thị** nó lên giao diện danh sách file ở góc trái.

---

### 2. Hai cách đơn giản nhất để anh tải file APK về điện thoại:

#### 👉 CÁCH 1: Xuất trực tiếp từ thanh công cụ AI Studio (Khuyên dùng)
1. Ở góc trên cùng bên phải giao diện Google AI Studio (hoặc trong phần Cài đặt của trình giả lập điện thoại đang phát trực tiếp), anh sẽ thấy biểu tượng **Settings (Răng cưa)** hoặc mục **Export**.
2. Nhấp vào đó và chọn **"Generate APK/AAB"** hoặc nút **Tải xuống APK**.
3. Hệ thống của AI Studio sẽ gửi trực tiếp đường dẫn tải file APK sạch về máy tính/điện thoại của anh.

#### 👉 CÁCH 2: Giải nén file ZIP mã nguồn
Em đã lưu sẵn file APK chất lượng cao tại thư mục gốc của dự án. Dự án này khi không có APK chỉ nặng **470 KB**, nhưng khi chứa APK sẽ nặng khoảng **12 MB**.
1. Anh bấm vào nút **"Export as ZIP"** (hoặc biểu tượng tải ZIP trong AI Studio) để xuất toàn bộ dự án này thành file `me-va-be.zip` về máy.
2. Anh kiểm tra dung lượng file tải về: Nếu file nặng khoảng **11.5 MB ~ 12 MB** thì tức là file APK đã nằm trọn vẹn ở bên trong.
3. Giải nén file `.zip` vừa tải về ra máy tính hoặc điện thoại, anh sẽ thấy file **`Install_Me_Va_Be.apk`** xuất hiện ngay ở thư mục giải nén để anh cài đặt bình thường!
